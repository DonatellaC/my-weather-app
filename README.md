## `Instructions`

Create a weather web application using:

- React
- Elastic UI framework
- API from: https://www.el-tiempo.net/api

and displaying:

- The current temperature and the probability of precipitation of a city in the province of Barcelona

### Getting started

- Install React
- Install Elastic UI framework

### Dependencies

Run `yarn install` to install React and `yarn start` to run servers on port 3000.
